# 具有延迟加载特性的 echarts vue 组件

```
组件属性包括：

option: echarts 配置

isLoading: 若置为 true，则会显示加载提示

resizeSignature: 可通过该值的改变触发组件的 resize 

maps: 想要注册的地图组件，格式为对象数组，对象包括：name: 注册名，data: 地图的 json 数据

scrollDomId: 用于绑定监听事件的 DOM 元素 ID 值，默认为 window
```
